# AeroPoint Support

AeroPoint is a local-network remote mouse and keyboard for a user-owned Mac or Windows PC.

## Requirements

- iPhone or iPad running iOS/iPadOS 17 or later.
- AeroPoint companion agent installed on your Mac or Windows PC.
- Both devices connected to the same Wi-Fi or local network.
- macOS users must grant Accessibility permission to the desktop agent.

## Pairing

1. Start the AeroPoint companion agent on your computer.
2. Open AeroPoint on your iPhone or iPad.
3. Scan the QR code shown by the desktop agent.
4. If QR pairing is unavailable, choose Manual and enter the IP address, port, and pairing nonce shown by the desktop agent.

## Troubleshooting

- Confirm both devices are on the same local network.
- Allow Local Network permission for AeroPoint in iOS Settings.
- Allow Camera permission if using QR pairing.
- On macOS, enable Accessibility permission for AeroPoint Agent in System Settings.
- Check that local firewalls allow the desktop agent to listen on port `41074`.

## Privacy

AeroPoint does not use accounts, cloud servers, analytics, ads, or tracking. Pairing data is stored locally on your device.
